Этот хук нужен для обновления содержимого [компонента](Компоненты), когда в нем меняются данные.
```
useEffect(() => {
  fetch(`http://127.0.0.1:8000/api-timetable/timetable/${group}/`)
	.then(response => response.json())
	.then(data => {
		setData(data.timetable);
	})
})
```

В этом случае мы меняем содержимое [компонента](Компоненты), когда к нам приходят данные с API.

Компонент перерисовывает каждый раз, когда в нем что-то меняется, чтобы отрисовать его единожды следует передать `[]` вторым аргументом, чтобы получилось:
```
useEffect(() => {
	// Code here
}, [])
```
